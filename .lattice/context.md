# Personal Site Mission Asimov

## Purpose

Tracks the full lifecycle of Julianna's personal website — from research and branding through design, build, content population, and the daily-update Claude skill. The quality target is Asimov Collective level: restrained, typographic, philosophical, alive. The site bridges architecture practice and software/AI practice into one unified identity artifact.

**PRD:** `PRD.md` at project root. Source of truth for scope, phasing, and success metrics.

## Conventions

- **Task IDs** follow project code `ASMV` (e.g., ASMV-1, ASMV-2)
- **Tasks map 1:1 to PRD phases** — ASMV-1 through ASMV-9 cover Phase 0 through Phase 6
- **Research deliverables** land in `research/` with numeric prefix matching their phase
- **Workflow:** Classic preset — `backlog → in_planning → planned → in_progress → review → done`
- **WIP limits:** 10 in_progress, 5 in review
- **Pre-code convention:** Tasks in Phase 0–2 (research, branding) produce markdown deliverables, not code. Implementation tasks begin at Phase 3 (ASMV-6: Design & build).
