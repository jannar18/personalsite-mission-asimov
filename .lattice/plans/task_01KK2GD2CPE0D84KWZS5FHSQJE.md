# ASMV-43: Add mobile navigation pattern (nav unreachable on touch after hero)

Header.tsx nav opens on hover only (handleMouseEnter). Touch devices cannot hover. Cross button is tappable (40x40) but at 25% opacity with no label — not discoverable. Mobile users past the hero section have no way to navigate except scrolling to the footer. Need a mobile-friendly nav pattern (hamburger, bottom nav, or improved cross affordance).
